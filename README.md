# NetCalcServer
This is a web API that solves mathematical expressions and requires user authentication with JWT tokens. The project is purely educational.

## Features:
 - mathematical expression evaluation
 - JWT-based authentication (currently under development)
 - Password hashing with bcrypt
 - Admin panel for user management
 - MySQL/MariaDB for user data storage

Client application available at https://github.com/sw229/NetCalcClient
